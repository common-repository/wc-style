== Description ==

This Plugin performs following functions And Developed by <a href="http://themelocation.com/">themelocation</a>:

1- Change WooCommerce Add to cart button color.<br>
2- Change WooCommerce Add to cart text color.<br>
3- Change WooCommerce text size on add to cart button.<br>
4- Change WooCommerce font style of button.<br>

<iframe width="560" height="315" src="https://www.youtube.com/embed/rCwLqYTjq7c" frameborder="0" allowfullscreen></iframe>

This plugin works on whole WooCommerce store product pages , shop page, cart & checkout page.

== Installation ==

1. From the Dashboard:
1. Plugins
1. Add New
1. Search "wc style"
1. Install Now
1. Activate
1. Go to Dashboard -> WC Style
